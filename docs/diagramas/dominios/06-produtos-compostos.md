# 06 — Produtos Compostos

```mermaid
flowchart TD
    A[Produto Comercial] --> B[Produto Simples]
    A --> C[Produto Composto]
    C --> D[Componentes Internos]
    C --> E[Kits ou Conjuntos]
    C --> F[Acabamentos]
    C --> G[Grupos de Composição]

    D --> H[Controle Operacional]

    F --> H
```
