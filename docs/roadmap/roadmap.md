# 🚫 Fora do Escopo Atual

O objetivo do projeto não é se tornar um ERP administrativo completo.

Portanto, neste momento não fazem parte do escopo:

* Contas a pagar
* Gestão de fornecedores
* DRE
* Financeiro corporativo completo
* ERP administrativo completo

---

# 🧠 Visão de Evolução

O sistema continuará evoluindo conforme as necessidades operacionais da Mobel Festas evoluírem.

A prioridade permanece sendo:

```text
Organização
   ↓
Operação
   ↓
Controle
   ↓
Eficiência
```

Cada nova funcionalidade deve possuir aplicação prática e valor direto para a operação da empresa.
